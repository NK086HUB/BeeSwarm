# 🎮 Настройка Unity проекта для Bee Swarm

## Требования
- **Unity версия:** 2022.3 LTS или новее
- **Платформа:** Windows, macOS, или Linux
- **Память:** 8+ GB RAM
- **Диск:** 10+ GB свободного места

## Шаг 1: Установка Unity

### Вариант A: Unity Hub (рекомендуется)
1. Скачайте [Unity Hub](https://unity.com/download)
2. Установите Unity Hub
3. В Unity Hub добавьте версию **2022.3 LTS**
4. Установите компоненты:
   - **Unity Editor** (обязательно)
   - **Windows/Mac/Linux Build Support** (в зависимости от ОС)
   - **Android/iOS Build Support** (опционально)

### Вариант B: Прямая установка
1. Скачайте Unity 2022.3 LTS с [официального сайта](https://unity.com/download)
2. Запустите установщик
3. Выберите компоненты как выше

## Шаг 2: Клонирование проекта

```bash
# Клонируйте репозиторий
git clone git@github.com:NK086tme/BeeSwarm.git
cd BeeSwarm

# Установите Git LFS (если ещё не установлен)
git lfs install

# Загрузите LFS файлы (если будут)
git lfs pull
```

## Шаг 3: Открытие проекта в Unity

1. **Запустите Unity Hub**
2. **Нажмите "Open"** → выберите папку `BeeSwarm`
3. **Дождитесь** импорта проекта (может занять несколько минут)

## Шаг 4: Настройка проекта

### Настройки Player:
1. **File** → **Build Settings** → **Player Settings**
2. **Company Name:** Введите ваше имя/название компании
3. **Product Name:** "Bee Swarm"
4. **Version:** 0.1.0

### Настройки качества:
1. **Edit** → **Project Settings** → **Quality**
2. Установите настройки для 3 уровней:
   - **Low** (для слабых ПК)
   - **Medium** (рекомендуемые)
   - **High** (для мощных ПК)

### Настройки ввода:
1. **Edit** → **Project Settings** → **Input Manager**
2. Проверьте стандартные настройки (должны подойти)

## Шаг 5: Структура проекта в Unity

### Папки которые должны быть:
```
Assets/
├── Scripts/           # C# скрипты
├── Scenes/           # Сцены игры
├── Prefabs/          # Префабы объектов
├── Materials/        # Материалы и шейдеры
├── Textures/         # Текстуры
├── Models/           # 3D модели
├── Audio/            # Звуки и музыка
├── Animations/       # Анимации
├── UI/               # Элементы интерфейса
└── Resources/        # Ресурсы для загрузки во время выполнения
```

## Шаг 6: Создание первой сцены

1. **File** → **New Scene**
2. **File** → **Save Scene** как `Main.unity` в `Assets/Scenes/`
3. Добавьте базовые объекты:
   - **Directional Light** (освещение)
   - **Camera** (камера)
   - **Empty GameObject** → назовите `GameManager`
   - **Empty GameObject** → назовите `HiveManager`

## Шаг 7: Настройка скриптов

### Добавление компонентов:
1. Выберите `GameManager` в иерархии
2. **Add Component** → **GameManager** (из папки Scripts/Core)
3. Выберите `HiveManager`
4. **Add Component** → **HiveManager**

### Настройка префаба пчелы:
1. **Create** → **3D Object** → **Cube** (временная модель)
2. Переименуйте в `Bee`
3. **Add Component** → **BeeController**
4. **Add Component** → **Rigidbody**
5. Перетащите `Bee` в папку `Assets/Prefabs/`
6. Удалите `Bee` из сцены

## Шаг 8: Тестирование

### Запуск игры:
1. **Нажмите Play** в редакторе Unity
2. Проверьте:
   - Работает ли UI отладки
   - Можно ли добавить пчёл (кнопка в UI)
   - Двигаются ли пчёлы

### Отладка:
- Используйте **Console** для просмотра логов
- Используйте **Scene View** для визуальной отладки
- Используйте **Inspector** для изменения параметров во время выполнения

## Шаг 9: Сборка

### Тестовая сборка:
1. **File** → **Build Settings**
2. **Add Open Scenes** (добавьте текущую сцену)
3. Выберите платформу (PC, Mac & Linux Standalone)
4. **Нажмите Build**
5. Выберите папку для сборки

## Частые проблемы и решения

### Проблема: Скрипты не отображаются
**Решение:** Перезапустите Unity или обновите проект

### Проблема: Ошибки компиляции
**Решение:** Проверьте Console и исправьте ошибки

### Проблема: Git LFS файлы не загружаются
**Решение:** Выполните `git lfs pull` в терминале

### Проблема: Низкая производительность
**Решение:** Уменьшите настройки качества в Unity

## Полезные горячие клавиши

- **F5:** Запуск/остановка игры
- **Ctrl+S:** Сохранить сцену
- **Ctrl+Shift+B:** Открыть Build Settings
- **Ctrl+0:** Сфокусироваться на сцене
- **Ctrl+1:** Сфокусироваться на иерархии
- **Ctrl+2:** Сфокусироваться на проекте
