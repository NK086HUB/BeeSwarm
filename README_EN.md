# 🐝 Bee Swarm - Unity Strategy Simulator

**Manage a bee colony in a realistic strategy-simulation game!**

## 🎮 About the Project
Bee Swarm is a strategy/simulation game where you control a bee colony with collective intelligence. Choose professions for bees, manage resources, survive seasons, and protect the hive from enemies.

## 🚀 Features
- **Professional Specialization** - 5 unique bee professions
- **Seasonal Mechanics** - 4 seasons with different tasks
- **Resource System** - 3 types of honey, pollen, special resources
- **Scouting & Defense** - fog of war, patrol system
- **Realistic Biology** + gameplay conventions
- **Optimization** - up to 500 bees with quality graphics and physics

## 🛠️ Technologies
- **Engine:** Unity 2022.3+
- **Language:** C#
- **Version Control:** Git + GitHub + Git LFS
- **Optimization:** LOD, Object Pooling, Smart Updates
