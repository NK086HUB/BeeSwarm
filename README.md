# 🐝 BeeSwarm

Симуляция поведения пчелиного роя. Агентное моделирование на Python с перспективой переноса в Unity.

## Запуск

```bash
cd src
python3 bee_basic.py
```

## Структура

- `src/` — исходный код
- `data/` — данные симуляции
- `models/` — ML модели
- `visualization/` — визуализация
- `docs/` — документация

## Статус

✅ Базовая симуляция работает: 60 пчёл, сбор нектара, возврат в улей
🚧 В разработке: Unity 3D, ML для поведения, Blender модели
