# 🐝 Bee Swarm - Unity Strategy Simulator

**Управляй пчелиным роем в реалистичной стратегии-симуляторе!**

## 🎮 О проекте
Bee Swarm - это игра в жанре стратегия/симулятор, где вы управляете пчелиным роем с коллективным интеллектом. Выбирайте профессии для пчёл, управляйте ресурсами, переживайте сезоны и защищайте улей от врагов.

## 🚀 Особенности
- **Профессиональная специализация** - 5 уникальных профессий пчёл
- **Сезонная механика** - 4 времени года с разными задачами
- **Система ресурсов** - 3 вида мёда, пыльцы, спецресурсы
- **Разведка и защита** - туман войны, патрулирование охранников
- **Реалистичная биология** + игровые условности
- **Оптимизация** - до 500 пчёл с качественной графикой и физикой

## 🛠️ Технологии
- **Движок:** Unity 2022.3+
- **Язык:** C#
- **Версионный контроль:** Git + GitHub + Git LFS
- **Оптимизация:** LOD, Object Pooling, Smart Updates

## 🚀 Начало работы
1. Клонируй репозиторий: `git clone https://github.com/NK086HUB/BeeSwarm.git`
2. Установи Git LFS: `git lfs install`
3. Открой проект в Unity 2022.3+
4. Начни с изучения документации в `memory/bee_swarm_project.md`
