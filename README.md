# 🐝 BeeSwarm — 2D

2D симуляция поведения пчелиного роя. Агентное моделирование.
В перспективе — перенос в Unity 2D.

## Запуск
```bash
cd src
python3 bee_basic.py
```

## Статус
✅ Базовая симуляция (Python): 60 пчёл, сбор нектара, возврат в улей
🚧 Unity 2D: перенос логики на C#, спрайты, анимации
🚧 ML: обучение поведения через нейросети (PyTorch)
